## 摘要
小样本目标检测旨在仅从少量标注样本中检测新目标类别，避免昂贵的大规模标注。近年来，基于原型的相似性学习方法通过将查询特征与类别原型匹配，实现了无需训练的适应。然而，它们存在两个根本限制：(i) 由类间相似性边界崩溃引起的类别混淆，以及 (ii) 精确定位缺乏足够的视觉线索，因为相似性分数仅捕获类别级语义亲和度，提供有限的空间信息。为解决这些问题，我们引入了两个互补组件。文本锚定语义掩码（TSMa）利用类别级文本特征作为语义锚点，通过视觉与文本特征间的通道交互来识别语义对齐的通道。通过抑制风格诱导的虚假响应并强化类别内在信号，TSMa扩大了类间相似性边界并减轻了类别混淆。我们进一步提出了阶段对齐的分层自回归回归（SHARe），它将定位重新表述为一个分层自回归过程，跨多个阶段逐步细化边界框。SHARe通过将特征抽象级别与回归阶段对齐，利用ViT表示的逐层特性：深层特征指导早期粗略定位，而富含边缘和纹理线索的浅层特征在后期阶段细化空间细节。在COCO上的实验展示了新的最佳水平，比之前最佳结果提升了+10.1 nAP，并通过广泛分析验证了每个组件。代码见 https://github.com/VisualScienceLab-KHU/ReSet。

## Abstract
Few-shot object detection aims to detect novel object categories from only a few labeled examples, avoiding costly large-scale annotation. Recent prototype-based similarity learning approaches enable training-free adaptation by matching query features with class prototypes. However, they suffer from two fundamental limitations: (i) class confusion arising from inter-class similarity margin collapse, and (ii) insufficient visual cues for precise localization, as similarity scores capture only class-level semantic affinity while providing limited spatial information. To address these issues, we introduce two complementary components. Text-Anchored Semantic Mask (TSMa) leverages class-level text features as semantic anchors to identify semantically aligned channels through channel-wise interaction between visual and text features. By suppressing style-induced spurious responses and emphasizing class-intrinsic signals, TSMa enlarges inter-class similarity margins and mitigates class confusion. We further propose Stage-Aligned Hierarchical Autoregressive Regression (SHARe), which reformulates localization as a hierarchical autoregressive process that progressively refines bounding boxes across multiple stages. SHARe leverages the layer-wise characteristics of ViT representations by aligning feature abstraction levels with regression stages: deeper layers guide early coarse localization, while shallower layers rich in edge and texture cues refine spatial details in later stages. Experiments on COCO demonstrate a new state of the art, outperforming the previous best by +10.1 nAP, with extensive analysis validating each component. The code is available at https://github.com/VisualScienceLab-KHU/ReSet.

### 1. 论文的核心问题与整体含义

**研究背景与动机**
*   **小样本目标检测 (FSOD)**：旨在仅使用少量（如1、10、30张）标注样本，检测出图像中的新类别物体，以解决大规模数据标注成本高昂的问题。
*   **现有范式局限**：近年来，一种“基于原型的相似性学习”范式成为FSOD的新方向（如DE-ViT, PiDiViT）。该方法无需对模型进行微调，直接利用支持图像构建类别原型，再计算查询图像特征与原型的相似度来同时进行分类和定位。然而，这项研究指出该范式存在两个根本性瓶颈：
    1.  **类别混淆 (Class Confusion)**：视觉特征中不可避免地包含与类别无关的风格信息，这会导致不同类别原型之间的相似性边界崩溃（Margin Collapse），使得模型难以区分语义不相关但风格相似的类别。
    2.  **定位不准 (Imprecise Localization)**：分类所用的余弦相似度主要编码了类别级的语义亲和力，这严重缺乏物体边界、形状和尺度等精确定位所需的空间几何信息。

### 2. 论文提出的方法论

核心思想是通过两个互补的组件，分别解决上述两个瓶颈： **文本锚定语义掩码 (TSMa)** 解决类别混淆，以及 **阶段对齐分层自回归回归 (SHARe)** 解决定位不准。

**关键技术细节**
*   **文本锚定语义掩码 (TSMa)**
    *   **核心思想**：利用来自CLIP的文本特征作为“语义锚点”。文本特征是对类别语义的紧凑表示，天然地剥离了视觉风格信息。通过与文本特征进行通道级交互，可以识别出视觉特征中真正反映类别本质语义的通道，并抑制风格驱动的虚假响应通道。
    *   **算法流程**：
        1.  **特征映射**：学习一个非线性映射函数 `ψ`，将文本特征投影到与视觉特征（DINOv2）相同的特征空间。
        2.  **通道激活**：计算视觉特征与映射后的文本特征的逐元素乘积，数值高的通道代表被两者同时激活。使用K-Means（K=2）聚类将这些通道分为高/低激活两组，高激活组被视为语义对齐通道。
        3.  **TF-IDF评分**：为每个通道计算TF-IDF分数。高频激活（TF高）且在其他类别中罕见（IDF高）的通道被视为高判别性通道。
        4.  **可学习软掩码**：根据TF-IDF得分对通道排序，并应用一个基于Sigmoid函数的可学习软阈值掩码，保留高排名通道，生成最终的**语义原型**。该过程去除了风格噪声，放大了类间相似性边界。
*   **阶段对齐分层自回归回归 (SHARe)**
    *   **核心思想**：将一次性定位重构成一个由粗到细的多阶段自回归过程，并巧妙地将ViT不同层的视觉特征，按逆序注入到各个回归阶段，以补充相似度特征中缺失的空间线索。
    *   **算法流程**：
        1.  **分层特征聚合**：将ViT的24层特征图划分为`S`个组（如`S=5`），每个组内通过可学习权重聚合为一个阶段视觉线索 `Vs`。浅层组富含纹理、边缘等低层几何信息；深层组则包含高层语义和全局上下文。
        2.  **逆序注入自回归回归**：回归过程分为`S`个阶段。在早期阶段（如阶段1），注入深层视觉线索`VS`以指导粗粒度定位；在后续阶段，逐步注入更浅层的视觉线索（如阶段`S`注入`V1`）以细化边界细节。该策略符合“先有全局语义理解，再精修局部几何”的直觉。
        3.  **阶段输入**：每个阶段的输入不仅包含上一个阶段的目标嵌入，还拼接了上一阶段的掩码预测结果，形成一个自回归的递进过程。
